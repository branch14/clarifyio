# Clarifyio

A ruby wrapper and CLI for the Clarify API.

* http://clarify.io/

## Installation

Add this line to your application's Gemfile:

    gem 'clarifyio'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install clarifyio

## Usage

### Commandline

    clarifyio index <audiofile>
    clarifyio tree
    clarifyio search <query>
